## Primary sources

- **[STD]** O-RAN Alliance: Architecture Description (O-RAN.WG1.OAD)
  Available at: o-ran.org (free registration)
  Focus on: Near-RT RIC / Non-RT RIC split, xApp vs. rApp scope, SMO framework.

- **[STD]** O-RAN Alliance: WG2 (Non-RT RIC, A1 interface) specifications
  Key for: rApp design, policy-based control, ML model management.

- **[STD]** O-RAN Alliance: WG3 (Near-RT RIC, E2 interface) specifications
  Key for: xApp design, E2 service models, real-time control loop constraints.
